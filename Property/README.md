# Property Java Maven Project

This is a basic Java Maven project scaffolded for development.

## Structure
- `src/main/java/App.java`: Main application entry point
- `src/test/java/`: For unit tests
- `pom.xml`: Maven build configuration

## Build & Run
To build and run the project, use:
```
mvn compile
mvn exec:java -Dexec.mainClass=App
```
